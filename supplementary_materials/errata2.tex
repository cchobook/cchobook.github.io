            \Orig \ce{CH2O + MnO2 + 4H+ -> CO2 + 2Mn^{2+} + 3H2O}
            \Corr \ce{CH2O + 2MnO2 + 4H+ -> CO2 + 2Mn^{2+} + 3H2O}
            \Thx{匿名（利川市第五中学）}
        \item[第 302 页，注记]
            \Orig 环丙酮和环己酮的羰基吸收前者波数小……
            \Corr 环丙酮和环己酮的羰基吸收前者波数大，这是因为环丙酮环内张力大，羰基碳于环内 \ce{C-C} 键的 p 成分增加，\ce{C=O} 键的 s 成分增加，从而增强了 \ce{C=O} 键。
            \Thx{匿名}
    \end{Errata}
